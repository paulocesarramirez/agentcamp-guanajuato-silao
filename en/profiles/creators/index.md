# Track: Creators (Tech)

For technical profiles (engineers, students, developers). Hands-on focus.

## Learning goals
- Understand best practices to design and build AI agents
- Prototype solutions with accessible tools
- Prepare a minimal environment to experiment

## Prerequisites
- Laptop with up-to-date browser
- (Optional) VS Code + Git
- Azure subscription (provided for practice at the event)

## Environment setup
1. Update your browser
2. Create/confirm your Microsoft account (to redeem Azure at the event)
3. (Optional) Install VS Code and Git
